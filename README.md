# Mustafa Hadi's Portfolio - React & Tailwind CSS

## Features

- Modern design with Tailwind CSS & gradients
- Fully responsive for desktop, tablet, and mobile
- Professional sections:
  - Introduction
  - About Me (Education, Interests, Certifications, Awards)
  - Experience & Leadership
  - Skills (Frontend, Backend, Tools)
  - Featured Projects
  - Contact Info
- Animated background blobs
- Dark theme with vibrant accent colors
- Configurable via a single `portfolioData` object

---

## Technologies Used

- **React** (CRA)
- **Tailwind CSS**
- **PostCSS**
- **Lucide React** for icons
